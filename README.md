# Marija Petrova Travel Landing Page

Single-page MVP for a personal travel designer based in North Macedonia, built with Next.js, TypeScript, and Tailwind CSS.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- All page copy lives in `data/siteContent.ts` so Dutch localization can be added later.
- The Marija portrait is stored in `public/images/marija-petrova.png` and the destination images in `public/images/*.jpg` are sourced from Wikimedia Commons with footer attribution.
- The contact form is UI-only in this MVP and logs submitted data to the browser console.
- Placeholder testimonials are clearly marked and must be replaced before launch.
